# 🎯 바이럴 헌터 워크스테이션

카테고리별 바이럴 마케팅 타겟 작업 전용 웹앱

---

## 🚀 시작하기

### 1️⃣ 서버 시작

**방법 1: BAT 파일 사용 (권장)**
```
START.bat 더블클릭
```

**방법 2: 명령어 사용**
```bash
cd C:\Projects\marketing_bot\viral_hunter_web
python app.py
```

서버가 시작되면 자동으로 브라우저가 열립니다.

---

## 🌐 접속 URL

브라우저에서 다음 주소로 접속:
- **http://localhost:5000**
- http://127.0.0.1:5000

## 📚 사용 방법

### 홈 화면
1. 전체 통계 확인
2. 카테고리별 우선순위 확인
3. 원하는 카테고리의 **[→ 작업 시작]** 클릭

### 작업 화면

#### 왼쪽: 타겟 리스트
- **검색**: 키워드로 타겟 검색
- **체크박스**: 여러 타겟 선택
- **클릭**: 타겟 선택하면 오른쪽에 상세 표시

#### 오른쪽: 타겟 상세
- 타겟 정보 확인
- 💬 **AI 댓글 생성하기** 클릭
- 생성된 댓글 확인/수정
- ✅ **승인** / ⏭️ **건너뛰기** / 🗑️ **삭제**

#### 상단: 일괄 작업
- ☑️ **전체 선택**: 모든 타겟 선택/해제
- 💬 **일괄 댓글 생성**: 선택한 타겟들의 댓글 한 번에 생성
- ✅ **일괄 승인**: 선택한 타겟들 한 번에 승인
- ⏭️ **일괄 건너뛰기**: 선택한 타겟들 한 번에 건너뛰기

---

## 💡 작업 팁

### 개별 작업
1. 리스트에서 타겟 클릭
2. AI 댓글 생성
3. 댓글 확인/수정
4. 승인/건너뛰기/삭제
5. 다음 타겟 선택

### 일괄 작업 (효율적!)
1. 체크박스로 여러 타겟 선택
2. **일괄 댓글 생성** 클릭 → 모든 댓글 자동 생성
3. 각 타겟 클릭하며 댓글 확인
4. **일괄 승인** 클릭 → 한 번에 처리!

### 검색 활용
- 특정 키워드가 포함된 타겟만 보기
- 예: "산후" 검색 → 산후 관련 타겟만 표시

## ⚠️ 문제 해결

### "사이트에 연결할 수 없음"
1. `STOP.bat` 실행
2. `START.bat` 다시 실행
3. 브라우저 새로고침 (Ctrl + F5)

### 타겟 리스트가 안 보임
- 브라우저 새로고침 (Ctrl + F5)
- 또는 시크릿 모드로 열기

### 서버가 시작 안 됨
- Python이 설치되어 있는지 확인
- Flask가 설치되어 있는지 확인:
  ```bash
  pip install flask
  ```

### 포트 5000이 이미 사용 중
- `STOP.bat` 실행 후 다시 시작
- 또는 다른 프로그램이 5000번 포트를 사용 중인지 확인

## 📞 지원

문제가 발생하면:

2. 브라우저 콘솔 (F12) 확인
3. `STOP.bat` → `START.bat` 재시작
