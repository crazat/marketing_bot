## 1. Executive Summary

### 1.1 현재 시스템 평가

| 영역 | 완성도 | 평가 |
|------|--------|------|
| **백엔드 API** | 75% | 핵심 기능 완성, 일부 API 미완성 |
| **프론트엔드** | 85% | UI/UX 우수, 성능 최적화 필요 |
| **데이터 수집** | 90% | Pathfinder, Scraper 완성도 높음 |
| **데이터 활용** | 50% | 수집은 잘 되나 인사이트 도출 미흡 |
| **자동화** | 40% | 수동 작업 비중 높음 |

### 1.2 핵심 발견사항

#### 강점
- Pathfinder V3 LEGION: 8라운드 멀티소스 키워드 수집 (2,674줄의 정성)
- Cafe Spy: 한국 맞춤형 맘카페 리드 수집 (가장 높은 전환율)
- Battle Intelligence: 순위 예측, 하락 알림 자동화
- Gemini AI 통합: 약점 분석, 콘텐츠 생성, 댓글 자동화

#### 약점
- **전환 추적 부재**: 리드→매출 경로 측정 불가능
- **경쟁사 분석 미활용**: 약점 데이터 수집만 하고 저장 안 됨
- **자동화 규칙 미작동**: auto_approval_rules 데이터 0개
- **API 미완성**: Export, Instagram, Reviews AI 기능

## 3. 마케팅 관점 신규 기능 제안

### 3.1 Phase 7.0: 전환 추적 시스템 (최우선)

#### 왜 필요한가?
> **"키워드 A에서 100명의 리드가 왔는데, 실제 매출은 얼마인가?"**
> 현재 이 질문에 답할 수 없음.

#### 구현 내용
```
1. Lead Manager에 "전환 기록" 버튼 추가
   - 전환 유형: 상담예약 / 시술 / 결제
   - 매출 금액 입력
   - 전환 날짜 자동 기록

2. 전환 분석 대시보드
   - 키워드별 전환율: "청주 다이어트 한약" 15% vs "청주 한의원" 5%
   - 플랫폼별 ROI: Naver 100만원/10리드 vs Blog 50만원/20리드
   - 월별 전환 추이 그래프

3. 자동 알림
   - "고전환 키워드 '청주 다이어트'에 새 리드 유입" 알림
```

#### 기대 효과
- **ROI 측정**: 마케팅 예산 최적화 가능
- **키워드 우선순위 재조정**: 전환율 기반 S등급 재분류
- **플랫폼 전략 수립**: 고전환 플랫폼 집중 투자

---

### 3.2 Phase 7.1: 경쟁사 약점 자동 분석 활성화

#### 왜 필요한가?
> **"경쟁사가 '대기시간이 길다'는 불만이 많으면, 우리는 '예약 시스템' 마케팅을 해야 함"**
> 현재 분석 API는 있으나 결과가 저장되지 않음.

#### 구현 내용
```
1. /competitors/analyze-reviews 결과 자동 저장
   - competitor_weaknesses 테이블에 저장
   - opportunity_keywords 자동 생성

2. 대시보드에 "이번 주 약점 TOP 5" 위젯
   - 약점 유형: 서비스 / 가격 / 시설 / 대기시간 / 효과
   - 심각도: Critical / High / Medium / Low

3. 약점 기반 콘텐츠 추천
   - "경쟁사 '비싸다' 약점 → '합리적 가격' 키워드 추천"
   - 자동 블로그 아웃라인 생성
```

#### 기대 효과
- **차별화 전략**: 경쟁사 약점을 우리 강점으로
- **콘텐츠 방향 설정**: 약점 기반 블로그 주제 자동 추천
- **시장 기회 발견**: 미충족 수요 키워드 발굴

---

### 3.3 Phase 7.2: 댓글 자동화 시스템

#### 왜 필요한가?
> **"매일 50개 바이럴 콘텐츠에 댓글을 달아야 하는데, 수동으로는 한계가 있음"**

#### 구현 내용
```
1. 댓글 템플릿 라이브러리
   - 카테고리별: 다이어트 / 피부 / 통증 / 교정
   - 상황별: 질문형 / 후기형 / 비교형
   - AI 변형: 같은 의미, 다른 표현

2. 자동 승인 규칙
   - "점수 80점 이상 + 질문형 + Naver Cafe" → 자동 승인
   - "점수 60점 미만" → 검토 필요 표시

3. 댓글 성과 추적
   - 좋아요 수, 답글 수 추적
   - 클릭률 (UTM 기반)
   - 전환 기여도 분석

4. 스케줄링
   - 시간대별 자동 등록 (9시, 12시, 18시)
   - 일일 최대 댓글 수 설정
```

#### 기대 효과
- **운영 효율화**: 댓글 업무 80% 자동화
- **일관된 품질**: 템플릿 기반 브랜드 톤 유지
- **성과 측정**: 어떤 댓글이 효과적인지 분석

---

### 3.4 Phase 7.3: 통합 마케팅 대시보드

#### 왜 필요한가?
> **"한 화면에서 전체 마케팅 성과를 볼 수 없음"**

#### 구현 내용
```
1. KPI 카드
   ┌─────────────────────────────────────────────┐
   │  이번 달 신규 리드    전환된 리드    매출     │
   │      127개             18개       2,400만원  │
   │      (+23%)           (+15%)      (+30%)    │
   └─────────────────────────────────────────────┘

2. 키워드 성과 테이블
   | 키워드 | 순위 | 리드 | 전환 | 매출 | ROI |
   |--------|------|------|------|------|-----|
   | 청주 다이어트 한약 | 3위↑ | 45 | 8 | 800만 | 320% |
   | 청주 한의원 | 1위- | 32 | 3 | 300만 | 150% |

3. 플랫폼별 퍼널
   Naver: 50명 → 40명 → 15명 → 5명 (전환율 10%)
   Blog:  80명 → 50명 → 20명 → 8명 (전환율 10%)

4. 경쟁 현황
   - 우리 순위 vs 경쟁사 평균
   - 이번 주 발견된 경쟁사 약점
   - 추천 대응 키워드
```

#### 기대 효과
- **의사결정 속도 향상**: 한눈에 전체 현황 파악
- **데이터 기반 전략**: 감이 아닌 수치로 판단
- **경영진 보고 용이**: PDF 리포트 자동 생성

### 3.5 Phase 7.4: AI 콘텐츠 어시스턴트

#### 왜 필요한가?
> **"키워드는 많이 발굴되는데, 실제 콘텐츠 작성이 병목"**

#### 구현 내용
```
1. 블로그 초안 자동 생성
   - 키워드 선택 → 아웃라인 생성 → 초안 작성 (Gemini)
   - 경쟁사 콘텐츠 분석 반영
   - SEO 최적화 제안

2. 리뷰 응답 자동 생성
   - 긍정 리뷰: 감사 + 재방문 유도
   - 부정 리뷰: 사과 + 개선 약속 + 오프라인 연락 제안
   - 중립 리뷰: 추가 정보 제공

3. SNS 캡션 생성
   - 플랫폼별 최적화 (Instagram, TikTok)
   - 해시태그 자동 추천
   - A/B 테스트용 변형 생성

4. 이미지 분석 및 추천
   - Vision Analyst 연동
   - "이 이미지 스타일이 바이럴되고 있음" 분석
   - 촬영 가이드 제공
```

#### 기대 효과
- **콘텐츠 생산성 3배 증가**: 초안 자동화
- **일관된 브랜드 톤**: 템플릿 기반 품질 관리
- **SEO 최적화**: AI 기반 키워드 배치

---

### 3.6 Phase 7.5: 고객 여정 추적

#### 왜 필요한가?
> **"고객이 어떤 경로로 우리를 찾아왔는지 모름"**

#### 구현 내용
```
1. UTM 파라미터 자동 추가
   - 모든 링크에 UTM 태그
   - utm_source: naver_cafe, youtube, instagram
   - utm_campaign: 키워드명
   - utm_content: 댓글ID

2. 터치포인트 추적
   - 첫 접점: "청주 다이어트 한약" 검색 → 블로그 클릭
   - 두 번째: 네이버 플레이스 방문
   - 세 번째: 전화 문의
   - 전환: 시술 예약

3. 어트리뷰션 모델
   - First Touch: 첫 접점에 100% 기여
   - Last Touch: 마지막 접점에 100% 기여
   - Linear: 모든 접점에 균등 배분
   - 선택 가능한 모델로 ROI 분석

4. 리타겟팅 리스트 생성
   - "블로그 방문 후 전환 안 됨" → 리타겟팅 대상
   - "키워드 A 관심자" → 맞춤 광고 대상
```

#### 기대 효과
- **마케팅 기여도 측정**: 어떤 채널이 효과적인가
- **예산 최적화**: 고효율 채널에 집중 투자
- **리타겟팅**: 이탈 고객 재유입
