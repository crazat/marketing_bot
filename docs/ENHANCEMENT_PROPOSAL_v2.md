## 2. 고도화 방향

### 2.1 핵심 철학 전환

```
[AS-IS] 리드 = 연락처가 있는 잠재고객
         ↓
[TO-BE] 리드 = 우리 서비스에 관심/필요가 있는 콘텐츠 발생 지점
```

**새로운 정의:**
- **리드**: 마케팅 개입이 효과적일 것으로 예상되는 온라인 대화/게시글
- **목표**: 연락처 수집 ❌ → 적시 적소에 유용한 정보 제공 ✅

### 3.1 Lead Scoring → **Opportunity Scoring**

#### 현재 문제
- "리드"라는 용어 자체가 연락처 기반 영업을 암시
- 점수가 높아도 실질적 행동으로 연결되지 않음

#### 제안: 기회 점수 시스템

```python
class OpportunityScorer:
    """
    기회 점수 = 마케팅 개입 효과 예측

    구성 요소:
    1. 적시성 (Timeliness): 지금 개입하면 효과적인가?
    2. 적합성 (Relevance): 우리 서비스와 관련 있는가?
    3. 개입가능성 (Accessibility): 댓글/답변이 가능한가?
    4. 영향력 (Impact): 다른 사람도 볼 가능성이 있는가?
    """
```

| 구성 요소 | 평가 기준 | 점수 |
|----------|----------|------|
| **적시성** | 게시 후 경과 시간 | 0-25 |
| | - 24시간 이내: 25점 | |
| | - 3일 이내: 20점 | |
| | - 7일 이내: 10점 | |
| | - 7일 초과: 0점 | |
| **적합성** | 키워드 매칭 | 0-30 |
| | - 핵심 키워드 (한의원, 다이어트 등) | +10/개 |
| | - 지역 키워드 (청주) | +15 |
| | - 경쟁사 언급 | +5 |
| **개입가능성** | 플랫폼 특성 | 0-25 |
| | - 지식인/카페 (답변 가능): 25점 | |
| | - 블로그 (댓글 가능): 15점 | |
| | - SNS (제한적): 10점 | |
| **영향력** | 노출 잠재력 | 0-20 |
| | - 인기 게시글/댓글 다수 | +10 |
| | - 검색 상위 노출 예상 | +10 |

#### 액션 연동

```
점수 80+  → 🔴 즉시 대응 권장 (24시간 내)
점수 60-79 → 🟡 우선 검토 대상
점수 40-59 → 🟢 주간 리뷰 대상
점수 0-39  → ⚪ 모니터링만
```

```python
class AuthenticityAnalyzer:
```

#### 답변 템플릿 고도화

**AS-IS (자동 생성, 일률적)**
```
안녕하세요! 청주 규림한의원입니다. 다이어트 한약 문의 주셔서 감사합니다...
```

**TO-BE (맥락 반영, 개인화)**
```python
class ResponseTemplateEngine:
    """
    상황별 답변 템플릿

    고려 요소:
    - 질문 유형 (가격, 효과, 위치, 후기 등)
    - 플랫폼 특성 (지식인: 전문적 / 카페: 친근)
    - 기존 댓글 맥락 (이미 답변이 있는지)
    - 브랜드 톤앤매너
    """

    def generate_draft(self, opportunity: dict) -> str:
        # 질문 유형 파악
        question_type = self._classify_question(opportunity['content'])

        # 플랫폼별 톤 조정
        tone = self._get_platform_tone(opportunity['platform'])

        # 맥락 반영
        context = self._analyze_context(opportunity)

        # 초안 생성 (반드시 인간 검토 필요)
        return self._compose_response(question_type, tone, context)
```

```python
        ],
    }

    def detect_signals(self, content: str) -> dict:
        """
        Returns:
        {
            'seeking_info': True/False,
            'open_to_suggestion': True/False,
            'urgency_level': 'high' | 'medium' | 'low',

        """
```

#### 신호 기반 우선순위

| 신호 조합 | 해석 | 우선순위 |

| 정보 탐색 + 열린 태도 | 비교 검토 중 | 🟡 높음 |
